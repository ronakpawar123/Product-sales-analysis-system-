# 📊 Product Sales Analysis System

🚀 A **Business Intelligence & Retail Analytics Dashboard** built using Python to help retail chains analyze sales performance, manage inventory, and make data-driven decisions.

📄 Full Project Report: 

---

## 🔍 Overview

The **Product Sales Analysis System** is designed to solve common retail challenges like:

* Overstocking & wastage
* Understocking & lost sales
* Lack of insights from raw sales data

This system provides an **interactive dashboard** for branch managers and a **Power BI dashboard** for higher-level decision-making.

---

## 🎯 Key Features

### 🧑‍💼 Branch-Level Dashboard (Streamlit)

* 🔐 Secure Login System (branch-specific access)
* 📈 Sales Summary & KPIs
* 🔼 Top Selling Products
* 🔽 Least Selling Products
* 📊 Category-wise Distribution
* ⏰ Expiry Alerts for products
* 📦 Smart Stock Recommendation

### 📊 Stakeholder Dashboard (Power BI)

* 💰 Revenue & Profit Analysis
* 📉 Trend Analysis (Monthly / Category)
* 🏪 Multi-Branch Comparison
* 🎛 Interactive Filters (Month, Product)

---

## 🛠️ Tech Stack

**Programming Language**

* Python 🐍

**Libraries & Tools**

* Pandas (Data Processing)
* Streamlit (Web Dashboard)
* Plotly & Matplotlib (Visualization)
* Power BI (Business Intelligence)

**Data Storage**

* CSV Files

---

## ⚙️ System Architecture

The system follows a **3-layer architecture**:

1. **Data Layer** → CSV files (branch-wise data)
2. **Logic Layer** → Data processing using Pandas
3. **Presentation Layer** → Streamlit Dashboard

---

## 🧠 Core Functionalities

* Data Cleaning & Processing
* Monthly Sales Analysis
* Expiry Detection System
* Stock Prediction using past trends
* Revenue Calculation (Units × Price)

## 🖥️ How to Run

```bash
# Clone the repo
git clone https://github.com/your-username/product-sales-analysis.git

# Navigate to folder
cd product-sales-analysis

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run dashboard.py
```
